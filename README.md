# Weihung Claude Plugin

Claude Code plugin with tools for plugin development.

## Skills

| Skill | Trigger |
|-------|---------|
| `write-skill` | "Help me write a skill for..." |
| `write-plugin` | "Create a plugin for..." |

## Installation

```bash
# From GitHub
/plugin marketplace add weihung/weihung-claude-plugin
/plugin install weihung-claude-plugin@weihung

# From local path
/plugin install /path/to/weihung-claude-plugin
```
